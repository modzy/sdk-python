modzy.client module¶

The API client implementation.

class modzy.client.ApiClient(base_url, api_key)[source]¶

Bases: object

The API client object.

This class is used to interact with the Modzy API.

The API models, jobs, and results can be accessed using the models, jobs and results attributes:

client = ApiClient(base_url='https://path/to/api', api_key='my-api-key')
model = client.models.get('my-model')
job = client.jobs.submit_files('my-model', '1.0.0', {'input': './my-file.dat'})
result = client.results.block_until_complete(job)

  • secret_access_key (str) – The AWS Secret Access Key.

  • region (str) – The AWS Region.

  • source_name (Optional[str]) – The source name. Defaults to ‘job’.

Returns

The submitted Job instance.

Return type

Job

Raises

ApiError – A subclass of ApiError will be raised if the API returns an error status, or the client is unable to connect.

Example

job = client.jobs.submit_aws_s3('model-identifier', '1.2.3',
{
    'model-input-name-1': {
        'bucket': 'my-bucket',
        'key': '/my/data/file-1.dat'
    },
    'model-input-name-2': {
        'bucket': 'my-bucket',
        'key': '/my/data/file-2.dat'
    }
},
    access_key_id='AWS_ACCESS_KEY_ID',
    secret_access_key='AWS_SECRET_ACCESS_KEY',
    region='us-east-1',
)

get_first_outputs()[source]¶

Gets the first or only outputs found in this result.

This is useful for retrieving the outputs after submitting a job with only a single input source where you do not know or care about the source name. For example:

job = client.jobs.submit_files('my-model', '1.0.0', {'input': './my-file.dat'})
result = client.results.block_until_complete(job)
outputs = result.get_first_outputs()
output_data = outputs['output-file.dat']
